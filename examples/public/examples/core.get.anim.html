WA.get, Examples of Animation of nodes


There are 9 basic animations methods:
Every animation method has a speed (in milliseconds, or the values 'slow', 'medium', 'fast'), and a callback function, called when the animation ends, or when a special event happens.

- WA.get(filter).fadeIn(speed, callback): fade in the nodes. The fade in applies directly on the opacity of the node. If the node is hidden, not visible or so, it will become visible. Only 'display' and 'opacity' are modified during the operation.
- WA.get(filter).fadeOut(speed, callback): fade out the nodes. The fade out applies directly on the opacity of the node. If the node is visible, it will become invisible. Only 'display' and 'opacity' are modified during the operation.
- WA.get(filter).openV(speed, callback, height)
- WA.get(filter).closeV(speed, callback, height)
- WA.get(filter).openH(speed, callback, width)
- WA.get(filter).closeH(speed, callback, width)
- WA.get(filter).open(speed, callback, width, height)
- WA.get(filter).close(speed, callback, width, height)
- WA.get(filter).move(speed, leftinit, topinit, callback, leftend, topend)

There is also a generic animation method:
WA.get(filter).anim(speed, callback): will animate all the found nodes with the specified script.

Let's play with some objects:
function test1() { WA.get('#d1').closeV(1000); } function test2() { WA.get('#d1').openV(1000, null, 50); }
